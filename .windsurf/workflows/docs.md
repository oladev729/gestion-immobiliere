## Workflow de Documentation

Ce workflow permet de générer et maintenir une documentation professionnelle pour le projet ImmoGest.

### Étapes

1. **Analyser le projet** pour identifier les composants clés
2. **Générer le README** principal avec les informations essentielles
3. **Organiser la documentation** dans les dossiers appropriés
4. **Mettre à jour** les guides d'installation et d'utilisation

### Commandes

// turbo
npm run docs:generate
